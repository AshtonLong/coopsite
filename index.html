Impact

Selected Projects

Recon Automation

Automated the morning recon workflow to remove repetitive steps and reduce errors, saving operators time and creating a consistent report for leadership.

  • Problem: The manual recon report was time‑consuming and error‑prone.
  • Approach: Scripted collection, formatting, and distribution; added simple guardrails.
  • Result: 16.25 hours saved over the term; consistent output.
  • Stack: Python for everything, windows API to hook into outlook.

Semantic Ticket Search Engine

Built a search experience powered by AI embeddings to retrieve results by meaning rather than keywords, improving relevance for real‑world ticket queries.

  • Problem: Managers requested keyword search; I proposed a semantic approach.
  • Approach: Generated embeddings for ticket text; implemented similarity ranking.
  • Result: Demoed to coworkers and managers; the team is adopting the semantic design.
  • Stack: Python + JS for data prep and UI, coded my own custom vector based DB for speed and memory efficiency.
